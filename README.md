# SQL Part 2

This repository contains advanced SQL queries focused on real-world data analysis problems
using PostgreSQL. The queries demonstrate the use of window functions, aggregations, and
advanced filtering techniques.

## Key SQL Concepts Used
- Window Functions (RANK(), LAG(), PERCENT_RANK())
- Common Table Expressions (CTEs)
- Aggregations (SUM, AVG, COUNT)
- JOIN operations
- GROUP BY and filtering logic
- Time-based analysis using DATE_TRUNC and EXTRACT
